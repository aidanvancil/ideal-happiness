Shut The Box Game
-----------------
Rules: With 2/1 dice knock down all 9 domino tiles in order to win.
Once the sum of the remaining tiles is down to <=6 you will only use one
dice to finish the game.


### Time Frame
## -------------
Finished game within seven hours.
